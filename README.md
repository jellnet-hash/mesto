# Проектная работа Mesto

Интерактивная страница, на которой пользователь может редактировать профиль, добавлять новые карточки с изображениями, удалять их, ставить лайки и просматривать изображения в полноэкранном режиме.

## Функциональность

- редактирование профиля;
- добавление новых карточек;
- удаление карточек;
- постановка и снятие лайков;
- открытие изображения в попапе;
- закрытие попапов по кнопке, оверлею и клавише Esc;
- плавное открытие и закрытие попапов.

## Технологии

- HTML5
- CSS3
- JavaScript (ES6)
- Webpack
- Git

## Установка и запуск

Клонировать репозиторий:

bash git clone <ссылка-на-репозиторий>

Перейти в папку проекта:

bash cd mesto-project

Установить зависимости:

bash npm install

Запустить проект в режиме разработки:

bash npm run dev

Собрать проект:

bash npm run build
